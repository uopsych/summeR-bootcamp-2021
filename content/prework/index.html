Hello! Welcome to the 4th Annual UO Psych R Bootcamp. There are a few things we’d like you to do before we get started to make things run as smoothly as possible.

If possible, please try to complete this checklist before the first bootcamp session. Thank you!


  1. Make sure you have a Zoom account set up through University of Oregon. We will use the same Zoom link for each day of the bootcamp, and you will receive the link via email and slack.

  2. Join the Slack workspace. This how we will primarily be communicating with each other, and it is the best place to ask questions outside of bootcamp hours. We’ll be using the channel #bootcamp2021. You should have received an email invitation to join the SlackRs workspace (check your Junk/Spam inbox) – if you haven’t received an invitation, send Brendan an email at bcullen@uoregon.edu. When you join, please introduce yourself and say hello in the #bootcamp2021 channel!

  3. Join the RStudio Cloud workspace. When you click this link you will be prompted to create a free RStudio Cloud account. Follow the instructions and click “Join space” when prompted. You should find yourself inside a workspace called UO Psych Bootcamp 2021. Please don’t open any of the materials just yet – we’ll go through these together during the bootcamp.

  4. Review the Code of Conduct.

  5. Optional - Download Google Chrome if you don’t have it already. The materials on this website are best viewed with Chrome. We’ve found that using other browsers (Firefox, Safari, etc.) can lead to some technical difficulties.

Looking forward to seeing you all soon. Hopefully we’ll have you loving R in no time!

-Brendan
